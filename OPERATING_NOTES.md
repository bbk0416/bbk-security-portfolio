# 운영 메모

## 현재 버전 판단
- 공개 가능한 1페이지 웹 포트폴리오 초안
- 증빙 창고가 아니라 경력/프로젝트 요약 페이지
- 군 관련 상세정보와 직접 식별 개인정보는 배제

## 프로젝트 노출 우선순위
1. MCP-Guard
2. VulnSignal
3. Cyber Armory
4. Pain Radar

## 보수적 표현 원칙
- "MCP 취약점 발견"이라고 쓰지 않음
- "MCP 기반 Agent 운용 환경의 위험 조건 검증"으로 표현
- "코인/토큰 프로젝트"라고 쓰지 않음
- "테스트넷 기반 Web3 보안 인프라 MVP"로 표현
- "Cyber Armory"는 공격도구처럼 보이지 않게 "보안 운영 플랫폼 프로토타입"으로 표현

# BBK Security Portfolio Site

정적 웹 포트폴리오 초안입니다.

## 목적
- 헤드헌터/기업에 바로 모든 증빙을 보내지 않고, 공개 가능한 경력 요약만 제공
- 보안엔지니어 / 취약점관리 / 보안운영 중심 포트폴리오 색인 역할
- 프로젝트는 안전한 범위의 개요만 제공

## 공개 원칙
이 사이트에는 아래 자료를 넣지 않습니다.

- 주민번호
- 상세주소
- 휴대전화
- 개인 이메일
- 군 내부 민감정보
- KVE 증명서 원본
- 군 경력증명서 원본
- 장교 인사자력표
- 비공개 GitHub 레포 링크
- 실제 취약점 신고 원문
- 내부망/기관명/상세 시스템 정보

## 배포 방법
### GitHub Pages
1. 새 공개 레포 생성 예: `bbk-security-portfolio`
2. `index.html`과 `assets/style.css` 업로드
3. Settings → Pages → Deploy from branch
4. main branch / root 선택

### Netlify/Vercel
이 폴더를 그대로 업로드하면 됩니다.

## 수정할 것
- 연락처를 공개할지 결정
- MCP-Guard 공모전 결과 반영
- CVD·VDP 승인/참여 상태 업데이트
- 공개 가능한 프로젝트 링크가 생기면 추가
